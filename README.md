# Java Examples

Collection of various Java programs demonstrating basic concepts, algorithms, and patterns.

## Files

- `day.java` - Day of week using switch
- `dessort.java` - Descending sort
- `diamondpat.java` - Diamond pattern
- `evencount.java` - Even numbers count in array
- `fact.java` / `factorial.java` - Factorial calculation
- `grade.java` - Grading system
- `hollow.java` - Hollow square pattern
- `anagram.java` - Check anagram
- `armstrong.java` - Armstrong number check
- Array related: array, arrayaver, arraymax, arraymin, arraysum
- `ascensort.java` - Ascending sort
- `char.java` - Print characters
- `conscount.java` - Consonant count
- `count.java` - Digit count
- `countuplow.java` - Upper and lower case count

## How to run

Compile and run each Java file individually using `javac` and `java`.
